# 1. 项目说明

本文面向小样本工业故障诊断中的故障样本稀缺问题，构建双流扩散增强模型 DSDM。模型采用“训练池预训练—5-shot 微调—增强样本生成—分类器验证”的实验流程。

DSDM 包含以下模块：

- 故障特征流；
- 故障语义流；
- 交叉注意力交互模块；
- 条件扩散解码模块；
- 故障特征频率保真损失。

实验使用公开工业故障数据，构建 5 类设备状态识别任务：

| 类别编号 | 类别名称 |
|---|---|
| 0 | 正常状态 |
| 1 | 轴承内圈故障 |
| 2 | 轴承外圈故障 |
| 3 | 齿轮磨损 |
| 4 | 转子不平衡 |

主实验设置为：

```text
任务形式：5-way 5-shot
每类真实支持样本数：5
每类生成增强样本数：200
扩散步数：200
重复实验次数：5
随机种子：2024, 2025, 2026, 2027, 2028
```

# 5. 数据预处理

## 5.1 先时序分块，后滑窗采样

为避免数据泄露，本文不采用“先滑窗后随机划分”的方式。

正确流程为：

```text
原始连续信号
      ↓
按时间顺序划分训练块、验证块、测试块
      ↓
训练块、验证块、测试块之间设置隔离间隔
      ↓
仅在各自数据块内部滑窗采样
      ↓
保存每个窗口对应的原始采样点索引
      ↓
进行采样点索引交叉校验
```

---

## 5.2 数据预处理命令

```bash
python src/data/preprocess.py \
    --raw_dir data/raw \
    --save_dir data/processed \
    --fs 12000 \
    --window_size 2048 \
    --stride 1024 \
    --train_ratio 0.8 \
    --val_ratio 0.1 \
    --test_ratio 0.1 \
    --gap 2048
```

参数说明：

| 参数 | 含义 |
|---|---|
| `--raw_dir` | 原始数据目录 |
| `--save_dir` | 预处理后数据保存目录 |
| `--fs` | 采样频率 |
| `--window_size` | 滑窗长度 |
| `--stride` | 滑窗步长 |
| `--train_ratio` | 训练块比例 |
| `--val_ratio` | 验证块比例 |
| `--test_ratio` | 测试块比例 |
| `--gap` | 时序块之间隔离间隔 |

# 7. 故障特征频率计算

## 7.1 理论频率参数

```text
采样频率 fs = 12000 Hz
转速 n = 1500 r/min
转频 fr = 25 Hz
滚动体数 Z = 8
滚动体直径与节径比 d/D = 0.206
接触角 alpha = 0°
齿轮齿数 z = 32
```

---

## 7.2 理论计算公式

轴承内圈故障特征频率：

```text
BPFI = Z/2 × fr × (1 + d/D × cos(alpha))
```

轴承外圈故障特征频率：

```text
BPFO = Z/2 × fr × (1 - d/D × cos(alpha))
```

齿轮啮合频率：

```text
GMF = z × fr
```

---

## 7.3 执行命令

```bash
python src/data/frequency_analysis.py \
    --data_dir data/processed/train \
    --fs 12000 \
    --rpm 1500 \
    --num_balls 8 \
    --diameter_ratio 0.206 \
    --contact_angle 0 \
    --gear_teeth 32
```

---

## 7.4 期望输出

```text
Rotating frequency fr: 25.00 Hz
BPFI: 120.60 Hz
BPFO: 79.40 Hz
GMF : 800.00 Hz
Rotor imbalance frequency: 25.00 Hz
```

对应论文中的理论值：

| 故障类型 | 理论频率 |
|---|---:|
| 轴承内圈故障 | 120.6 Hz |
| 轴承外圈故障 | 79.4 Hz |
| 齿轮磨损 | 800.0 Hz |
| 转子不平衡 | 25.0 Hz |

# 24. 注意事项

1. 测试集不得参与 DSDM 预训练、微调、增强样本筛选或分类器训练。
2. 归一化参数只能由训练集计算，不能使用验证集或测试集统计量。
3. 必须先按时间块划分数据，再在数据块内部进行滑窗采样。
4. 每次重复实验必须保存随机种子、支持集索引和生成样本索引。
5. 所有对照组必须使用相同的支持集、相同生成样本数量和相同分类器结构。
6. t-SNE 仅作为辅助可视化，不能作为生成质量的唯一依据。
7. 论文中报告的均值和标准差必须来自 5 次独立重复实验。
8. 若重新更换数据或划分方式，论文中的实验结果需要重新计算。

# 25. 项目核心结论对应代码输出

运行完整实验后，应能得到以下结论：

1. DSDM 在 1-D CNN 上取得最高分类准确率；
2. DSDM 在 SVM 上同样优于 Add、Concat 和 Linear 对照组；
3. DSDM 的 PSD-MSE、MMD、DTW、FD 均低于对照组；
4. DSDM 的 KS 检验通过率最高；
5. 消融实验表明余弦调度、多尺度特征流、语义流、交叉注意力和频率保真损失均有贡献；
6. 注意力权重主要集中在故障冲击或周期振动片段附近；
7. 数据泄露检查结果显示训练集、验证集和测试集无原始采样点重叠。
